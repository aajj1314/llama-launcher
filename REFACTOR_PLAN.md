# 项目重构计划

## 项目分析

### 项目功能
- llama.cpp 模型启动器
- 支持三种运行模式：CLI、Server、Embedding
- 配置参数管理（上下文大小、GPU层数、端口等）
- TUI 和 WebUI 界面
- 状态管理和监控
- 模块化架构
- 配置管理
- 进程管理

### 已解决问题
- WebUI 参数修改功能不稳定 - 已修复
- 前端和后端同步问题 - 已修复
- 错误处理不完善 - 已完善
- 代码结构需要优化 - 已优化

## 重构架构（已实现）

### 核心模块
1. **配置管理模块** (`config.py`)
   - 配置常量定义
   - 环境变量支持
   - 配置加载和管理

2. **状态管理模块** (`state_manager.py`)
   - 单例模式状态管理
   - 配置持久化
   - 线程安全
   - 状态同步机制

3. **进程管理模块** (`process_manager.py`)
   - 模型进程启动和管理
   - 不同运行模式支持
   - 进程参数构建

4. **工具模块** (`utils.py`)
   - 日志解析
   - 进程终止
   - 通用工具函数

5. **后端 API 模块** (`web_app.py`)
   - FastAPI 后端
   - RESTful API 设计
   - 错误处理和验证
   - 实时指标监控

6. **TUI 模块** (`run.py`)
   - 终端用户界面
   - 模型管理
   - 实时状态显示
   - 赛博朋克风格

7. **WebUI 模块** (`templates/index.html`)
   - Vue.js 前端
   - 响应式设计
   - 实时状态同步
   - 远程管理界面

8. **启动器模块** (`launcher.py`)
   - 统一启动脚本
   - 模式切换（TUI、WebUI、两者同时）
   - 命令行参数解析
