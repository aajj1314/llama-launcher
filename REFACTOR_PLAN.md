# 项目重构计划

## 项目分析

### 项目功能
- llama.cpp 模型启动器
- 支持三种运行模式：CLI、Server、Embedding
- 配置参数管理（上下文大小、GPU层数、端口等）
- TUI 和 WebUI 界面
- 状态管理和监控

### 现有问题
- WebUI 参数修改功能不稳定
- 前端和后端同步问题
- 错误处理不完善
- 代码结构需要优化

## 重构架构

### 核心模块
1. **状态管理模块** (`state_manager.py`)
   - 单例模式状态管理
   - 配置持久化
   - 线程安全

2. **后端 API 模块** (`web_app.py`)
   - FastAPI 后端
   - RESTful API 设计
   - 错误处理和验证

3. **TUI 模块** (`run.py`)
   - 终端用户界面
   - 模型管理
   - 实时状态显示

4. **WebUI 模块** (`templates/index.html`)
   - Vue.js 前端
   - 响应式设计
   - 实时状态同步

5. **启动器模块** (`launcher.py`)
   - 统一启动脚本
   - 模式切换
